# 💾 XpLauncher — Windows 98 Edition

> Легковесный, полностью портативный офлайн-лаунчер для Minecraft, выполненный в классической и атмосферной стилистике операционной системы Windows 98. 

Приложение построено на базе современного и производительного фреймворка **Tauri** (Rust + TypeScript), благодаря чему готовый билд весит всего **~10–15 МБ** и потребляет минимум системных ресурсов.

---

## ✨ Основные особенности

* **💾 Полная портативность:** Все файлы игры (`.minecraft`) и рантаймы Java скачиваются локально в папку с лаунчером. Программа не оставляет мусора в системе и может запускаться с обычной флешки.
* **⚡ Умное управление Java:** Благодаря интеграции с `lighty-java`, лаунчер сам определяет, какая версия Java нужна под выбранный клиент (Java 8 для 1.7.10/1.12.2, Java 17 для 1.17+, Java 21 для 1.21+), автоматически скачивает её и кэширует. Игроку не нужно устанавливать Java на ПК вручную.
* **⚙️ Классическое меню свойств:** Ретро-окно настроек в стиле Windows 98 позволяет гибко распределять оперативную память (JVM RAM args) и задавать разрешение игрового окна. Настройки сохраняются между запусками.
* **🖼️ Безрамочный интерфейс:** Стандартная рамка Windows 11 полностью убрана. Окно имеет собственную ретро-полосу заголовка, за которую его можно перетаскивать, а также рабочие кнопки сворачивания и закрытия.

---

## 🛠️ Стек технологий

* **Фронтенд:** HTML5, CSS3, TypeScript, библиотека стилей **98.css**.
* **Бэкенд:** Rust, фреймворк **Tauri v2**.
* **Ядро запуска Minecraft:** Крейт `mc-launcher-core` (Rust).
* **Менеджер Java:** Крейт `lighty-java` (Rust).

---

## 📥 Скачать и играть

Готовый скомпилированный файл лаунчера (`.exe`) можно найти в разделе **[Releases](../../releases)** на странице репозитория.

Для начала игры достаточно просто скачать этот файл и запустить его. Устанавливать ничего не нужно — лаунчер сам скачает необходимые версии Minecraft и Java прямо в ту папку, из которой вы его запустили.
